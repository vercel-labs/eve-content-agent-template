# X (Twitter) post specs

Quick-lookup specs for drafting and publishing on X. Default to free-account limits unless the
writer says the account has Premium.

## Character limits

| Surface | Limit |
| --- | --- |
| Post (free account) | 280 characters |
| Post (X Premium, any tier) | up to 25,000 characters |
| In-feed preview of a long post | ~280 characters, then "Show more" |
| Bio | 160 characters |
| Display name | 50 characters |
| Username / handle | 4–15 characters |
| Direct message | 10,000 characters |
| URL (auto-shortened) | counts as 23 characters, any length |

- Counts include spaces, punctuation, emojis, and hashtags.
- Free accounts can't create long posts; their text truncates at 280.
- Practical target: 240–259 characters tends to earn the most engagement (Sprout); keep the
  full point inside the first ~280 so the in-feed preview alone delivers value.

## Hashtags

- 0–2 real, relevant hashtags. 1–2 can lift engagement; 3+ tends to lower it. No hashtag walls.

## Media

- Native image or video lifts engagement; X favors native (especially video) over off-platform links.
- Add media only when it carries the point — text-only posts perform strongly on X.

## Links

- External links cut reach (Sprout: ~50–90% reduction). Put a needed link in the first reply and
  say so, rather than in the main post. Space multiple link posts out over time.

## Cadence

- Sustainable daily posting beats bursts; ~3–5 posts/day suits growth-focused accounts.
- Weekday mornings (~9–11 AM, Tue–Thu) skew highest on average — but post when your audience is on.
