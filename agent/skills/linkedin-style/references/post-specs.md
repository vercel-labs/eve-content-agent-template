# LinkedIn post specs

Concrete limits and specs for quick reference. Only the numbers below are sourced; don't add
others without a source.

## Text post

| Thing | Value |
| --- | --- |
| Individual post (status update) max | 1,300 characters |
| Company page status update max | 700 characters |
| Visible before "…see more" (mobile) | ~200 characters |
| Visible before "…see more" (desktop) | ~300 characters |

The truncation point is the practical budget for the hook: get the reader to click before the
fold. There is no published "ideal" length beyond the truncation threshold — write only as long
as the one idea needs.

## Document / carousel post

Carousels are uploaded as a **PDF** (image files like JPG/PNG won't work as a carousel).

| Thing | Value |
| --- | --- |
| File type | PDF |
| Page size | 1080×1080 (square) or 1920×1080 (landscape) |
| Max pages | < 300 |
| Max file size | 100 MB |

Carousel ads (paid, via Campaign Manager) are a different format with their own rules: max 10
cards plus a CTA button.

## Hashtags

- 3–5 per post, grouped at the end.
- CamelCase multi-word tags (`#SocialMediaMarketing`).
- Specific/niche over broad.
